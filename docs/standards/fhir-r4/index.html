FHIR R4

The platform uses FHIR R4 as the data standard, powered by HAPI FHIR 7.4.5.

Resources used​

ResourceUsed byPurpose
PatientPortal, ClientPatient selection and display
PractitionerAuth Server, IdPClinician identity (fhirUser)
ConditionSMART ClientClinical data display
MedicationRequestSMART ClientMedications display
ObservationSMART ClientLab results
ConsentConsent Manager (v1.1.0)Patient consent directives
AuditEventATNA Audit (v1.1.0)Immutable audit trail
ServiceRequestReferral Module (v1.2.0)Inter-facility referrals
TaskReferral Module (v1.2.0)Referral workflow state

HAPI FHIR version tracking​

The platform tracks HAPI FHIR stable releases. When HAPI ships, the platform follows within 2–4 weeks for minor releases and 4–6 weeks for major releases.
