Configuration Reference

immunization.*​

Bound from application.yml by ImmunizationProperties (@ConfigurationProperties(prefix = "immunization")). Validated at startup — missing @NotBlank fields prevent the app from starting.

PropertyTypeDefaultRequiredDescription
immunization.client-idStringaj-fhir-immunization✓OAuth2 client_id registered on the auth server
immunization.redirect-uriStringhttp://localhost:8084/callback✓Must match auth server registered_apps.redirect_uri exactly
immunization.scopesStringlaunch openid fhirUser patient/Patient.rs patient/Immunization.rs patient/ImmunizationRecommendation.rs✓Space-separated SMART scopes (multi-line YAML OK)
immunization.auth-server-urlStringhttp://localhost:9000✓Auth server base URL — used for SMART discovery
immunization.fhir-base-urlStringhttp://localhost:8080/fhir✓HAPI FHIR JPA base URL
immunization.token-refresh-buffer-secondslong120Seconds before expiry to trigger proactive token refresh
immunization.discovery-cache-ttl-secondslong600TTL for /.well-known/smart-configuration cache per ISS
immunization.who-schedule-pathStringclasspath:who-vaccine-schedule.jsonBundled WHO vaccine schedule (reserved for future schedule advisor)

spring.cache.*​

PropertyValueDescription
spring.cache.typecaffeineCache implementation
spring.cache.caffeine.specmaximumSize=500,expireAfterWrite=600sMax 500 entries, 10-minute TTL

spring.session.*​

PropertyDefaultDescription
spring.session.timeout3600sHTTP session timeout — matches access token default TTL
spring.session.store-type(none — in-memory)Set to redis for multi-instance deployments

server.*​

PropertyDefaultDescription
server.port8084App port. Platform map: 8080=HAPI, 8081=SMART Client, 8082=Consent, 8083=Referral
server.forward-headers-strategy(not set)Set to native when behind a reverse proxy for correct redirect URI generation

management.*​

PropertyValueDescription
management.endpoints.web.exposure.includehealth,info,metricsExposed actuator endpoints
management.endpoint.health.show-detailswhen-authorizedFull health details only for authenticated requests

logging.*​

PropertyDefaultDescription
logging.level.com.ajfhir.immunizationINFOApp log level — change to DEBUG for detailed FHIR query and SMART handshake logging
logging.level.org.springframework.securityWARNSecurity framework log level
logging.level.ca.uhn.fhirWARNHAPI FHIR client log level
