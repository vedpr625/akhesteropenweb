How to use this sandbox

1
Pick your entry point
Evaluator? Start with Clinical Scenarios — no technical knowledge required. Developer? Start with the API Console to test scope enforcement live.
2
Use the test credentials
Every page uses the same pre-seeded data. Clinician dr.smith, patient P. Kumar (ePatient-8675309), SMART client akhester-smart-client. No login required.
3
Change values, watch the platform react
Toggle consent permit → deny, step through lab and referral workflows, edit scope strings in the Console. Every action produces a real FHIR AuditEvent.
4
Check the ATNA audit trail
Every interaction writes a FHIR AuditEvent. The Scenarios audit trail accumulates for your session — click any row to expand the full IHE ATNA JSON.
